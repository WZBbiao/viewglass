# Viewglass Platform Plan

This repository is evolving from a Lookin-derived desktop inspector into a modern, AI-oriented iOS runtime inspection platform.

Current public deliverable:

- `viewglass`: a CLI-first runtime inspector compatible with LookinServer

Target product direction:

- AI-oriented iOS runtime inspector
- CLI-first UI introspection toolkit
- Lookin-compatible runtime inspection platform
- future skills and MCP support

## Recommended Brand

Recommended standalone product name:

- `Viewglass`

The repository now ships under the `Viewglass` brand and `viewglass` binary, while remaining compatible with the Lookin ecosystem where practical.

This project is independently maintained and built on top of the open-source Lookin ecosystem.

- compatible with `LookinServer` where practical
- no longer limited to the original macOS GUI product shape
- intended to support CLI, automation, skills, and MCP workflows
